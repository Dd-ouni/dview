# ddview
  
## 什么是ddview？
ddview 是对 element plus 基础组件进行增强，解决业务开发遇到的痛点，提高效率。element plus 提供的基础组件粒度可谓是非常细腻。这是它受欢迎的原因。可以很灵活的搭配。比如 Table 和 Pagination 是分离的。在非 Table 的分页场景单独使用 Pagination，但却给业务开发带来很多方面的痛点。
